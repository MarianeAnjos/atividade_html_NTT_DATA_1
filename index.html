Como funciona o HTML?

(voltar)

Em 1991, criado por Tim Berners-Lee, na Suiça, o HTML foi criado para fazer ligações entre instituições de pesquisa próximas para compartilhar documentos com mais facilidade.

O HTML não é uma linguagem de desenvolvimento como de inicio costumamos pensar, ele é uma linguagem de marcação, as linguagens de marcação serve, ára delmitar conteúdos especificos a partr de uma sintaxe própria. Com o HTML utilizamos códigos para defirinir tipos de letras, tamanho, cor, espaçamento, e outros demais aspectos do site.




O que são tags?

(voltar)

As tags no HTML são formadas por uma estrutura de linguagem prórpia para uma página na web, essa estrutura é formada return new Promise uma conjunto de elementos que se conectam formando uma página;

Eles informam ao navegador o tipo de estrutura que está sendo utilizada, seja um paráfrafol, um espaço, imagens e assim por diante.

Exemplos de tags:

  1. Tag h1: Utilizada para títulos, geralmente utiliza-se apenas uma por página
  2. TAG h2: Utilizada para subtitulos podendo ir até o h4 sofrendo alterações em seu tamanho
  3. TAG p: Utilizada para iniciar parágrafos
  4. TAG br: Utilizada para dar espaço no corpo do HTML

Obs * para utilizar as tags citadas acima é necessário abrir e fecha-las




Como funcionam os links no HTML?

(voltar)

Para utilizarmos os links no precisamos utlizar a tag a. Dentro desta tag utilizamos o atributo href que é o endereço de destino do Link.
